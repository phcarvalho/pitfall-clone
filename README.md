# 🐊 Pitfall! Clone

This is a clone of the classic **Pitfall!** from the Atari 2600, created for study purposes.

The project was built entirely using **Godot**, with assets recreated in **Aseprite** based on the original game. The goal is to replicate the look and feel of the original as closely as possible while learning game development concepts.

---

## ▶️ Play Now

You can play the game in your browser here:
- [Pitfall! Clone](https://phcarvalho.itch.io/pitfall-clone)

---

## 🕹️ Controls

* **← / → (Arrow Keys)** — Walk
* **Space** — Jump
* **↑ / ↓ (Arrow Keys)** — Climb stairs
